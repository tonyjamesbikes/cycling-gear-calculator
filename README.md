## Introduction

This project is a simple PHP application that demonstrates how to build a basic cycling gear calculator using PHP. The application allows users to plan their bicycle gear rations through a simple web interface.

## Installation

To install the application, follow these steps:

1. Clone the repository to your local machine.
2. Launch the application by navigating to the `index.php` file in your web browser.

## Usage

To use the application, follow these steps:

1. Launch the application by navigating to the `index.php` file in your web browser.
2. Enter your gearing details.
